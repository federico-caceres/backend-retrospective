# Retrospective project

Proyecto final en BootCamp de Make it Real

## Comenzando

### Prerequisitos

- Node.js instalado en el equipo
- Dependencias del proyecto instaladas (ejecutar `npm install`)
### Configuración

- Copiar el archivo `env.example` a `.env`

### Ejecución del proyecto

1. Ejecutar el comando `npm start`
2. Acceder a la aplicación en el navegador con la URL `http://localhost:3001`

### Postman Collection
`https://github.com/federico-caceres/backend-retrospective/blob/main/bck-retro.postman_collection.json`

### Tecnologías utilizadas

- Node.js
- Express
- MongoDB
